# Ayshata Guard

**Check before you connect.**

Ayshata Guard is an AI-powered onchain safety and scout network built to help crypto users review airdrops, testnets, campaigns, token launches, and suspicious links before risking their wallet.

The goal is simple: make Web3 participation safer by combining AI-assisted risk checks with community scout reports.

## What is Ayshata Guard?

Crypto users often find new opportunities through X posts, Telegram groups, Discord links, testnets, airdrops, and early token campaigns. Some are real. Some are low quality. Some are wallet drainers.

Ayshata Guard helps users slow down and check basic safety signals before taking action.

## MVP Features

### AI Risk Checker

A simple checker that reviews basic project context:

- Project name
- Website
- X/Twitter profile
- Chain
- Task or campaign instruction

The checker returns a practical sample report with:

- Risk score
- Red flags
- Positive signals
- Safer action suggestion
- Verdict

### Early Scout Waitlist

Ayshata Guard is also building an early scout network for people who want to help discover, test, and review Web3 opportunities.

Scout roles:

- **Scout** — finds fresh airdrops, testnets, launches, and campaign links
- **Tester** — tests app flows using safe wallet practices
- **Researcher** — verifies socials, docs, launch details, and risk signals

## Safety Principles

Ayshata Guard is safety-first.

We do not ask users for:

- Seed phrases
- Private keys
- Deposits
- Main wallet access

Recommended user behavior:

- Use burner wallets for unknown apps
- Never sign unclear transactions
- Verify official links before connecting
- Avoid sending funds to unverified campaigns
- Treat high-reward claims with caution
