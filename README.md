# Ayshata Guard

Ayshata Guard is an AI-powered onchain safety and scout network for checking airdrops, testnets, campaigns, and new token launches before users risk their wallet.

## Live Website

https://ayshataguard.netlify.app/

## MVP

- AI Risk Checker
- Early Scout Waitlist
- Scout / Tester / Researcher roles
- Safety-first Web3 campaign review

## Waitlist Fields

- X/Twitter handle: required
- Role: required
- Email: required
- EVM address: required

## Safety Rules

- No seed phrase
- No private key
- No deposit required
- Use burner wallets for testing unknown apps

## Tech Stack

- Static HTML/CSS/JavaScript
- Netlify hosting
- Netlify Forms for Early Scout submissions

## Deploy on Netlify

1. Push this repo to GitHub.
2. Open Netlify dashboard.
3. Add new site from Git.
4. Connect this repository.
5. Build command: leave empty.
6. Publish directory: `/`.
7. Deploy branch: `main`.
